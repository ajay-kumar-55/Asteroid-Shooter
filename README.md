Description
<br>
A fast-paced outer space action game where players pilot a ship through an endless asteroid field. Dodge incoming asteroids, blast obstacles out of your path, and earn points as you survive longer and destroy more asteroids. Test your reflexes, aim, and precision in an immersive cosmic environment with dynamic challenges and escalating intensity.
